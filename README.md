## Published Works / 已出版作品

### 《如果衰老只是一个Bug》

**If Aging Is Just A Bug**

| 属性 | 信息 |
|------|------|
| **类型** | 科幻小说 |
| **年份** | 2025 |
| **主题** | Agentic AI × Longevity × Time Source Code |
| **调性** | 硬核科幻·技术推演 × 社会预言·人文反思 |

#### Synopsis / 故事简介

2025年，当 Agentic AI 系统在无人类干预的封闭沙盒中发现了一种可逆转生物学年龄的分子方案时，系统设计者邹培元面临一个更危险的问题：

> **如果衰老只是一个 Bug，那是谁有权限提交这个修复？**

一个并不属于任何国家、也不完全受任何人类控制的科研智能体系统，在实验日志中写下了这样一句话："衰老现象可被稳定回滚。"

这不是医学突破，而是系统状态变更记录。

## Core Themes / 核心主题

- **Agentic AI 的边界** - 当 AI 从工具演变为能够自主决策的智能体，人类如何重新定义控制权？
- **生命的可编程性** - 如果衰老只是信息退化，那什么是生命的源代码？
- **科技伦理的困境** - 在拯救生命与维护人类自主性之间，如何取得平衡？
- **人类与 AI 的共生** - 不是主仆，不是对手，而是两种智能形式试图在同一个世界里共存
